pwd : 현재 디렉토리 확인 (경로를 보여줌)
~ : 홈 디렉토리 (c:/users/user) 
ls : 현재 위치 디렉토리 의 파일 및 폴더 확인
   -a : 숨김 파일 및 디렉토리 확인 옵션
   -l : 파일과 디렉토리의 상세정보 확인 옵션
clear : 터미널의 기록 지우기
cd : 디렉토리 이동
 ../ : 상위 디렉토리
 ./ : 현재 디렉토리
 mkdir : 디렉토리 생성

vim : 텍스트 에디터 실행
 - vim 파일명 ? 새로 생성 : 열기
 - i : 작성모드 / esc : 읽기모드
 - :w 작성 / :q 종료 / :wq 작성후 종료 /q! 강종 

rm : 파일 삭제
 -r : 디렉토리까지 삭제(하위 디렉토리 및 파일 모두 삭제)
